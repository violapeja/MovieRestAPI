The purpose of the application is to build a Rest API to manage movies.

Used technologies

  • Programming language: C#
  • Database: SQL Server
  • Frameworks: ASP.NET Web API, Entity Framework

Local environment setup

  • Restore database backup found at Database Backups folder of the project root folder
  • Update connectionString property of the connection string named MovieContext in the web.config file to match your local database
  • Compile the solution

How to test the application

  • First you need a Rest API Client for the test (e.g. Postman)
  • For getting all the movies
    • URI: /api/movies
    • HTTP METHOD: GET
  • For getting a single movie
    • URI: /api/movies/{movieId}
    • HTTP METHOD: GET
  • For posting a new movie
    • URI: /api/movies
    • HTTP METHOD: POST
  • For updating a movie
    • URI: /api/movies/{movieId}
    • HTTP METHOD: PUT
  • For deleting a movie
    • URI: /api/movies/{movieId}
    • HTTP METHOD: DELETE
  • For searching movies in a realise year and ordering by an attribute
    • URI: /api/movies/search/{year}/[title | director]
    • HTTP METHOD: GET
